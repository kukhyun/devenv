# DevEnv
Development Environment for Emacs, Linux, OSX  
Emacs, Linux, OSX를 위한 개발환경

## Version

* Ubuntu 16.04 LTS / 14.04 LTS
* Emacs 24.5.x

* markdown-mode
* elixir-mode

* powerline
